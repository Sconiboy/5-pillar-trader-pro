# UI Style Guide: Stitch Design System

## Color Palette
- **Background:** `#121212` (Deep Charcoal)
- **Surface/Cards:** `#1E1E1E`
- **Primary Accent:** `#00FF41` (Neon Green - Bullish)
- **Secondary Accent:** `#FF3B30` (Apple Red - Bearish)
- **Text Primary:** `#FFFFFF`
- **Text Secondary:** `#8E8E93`

## Typography
- **Headings:** Inter Bold
- **Stock Tickers:** Roboto Mono (Monospaced for alignment)
- **Body:** Inter Regular

## Component Specs
- **Border Radius:** 12px for cards, 30px for "Grade" badges.
- **Padding:** 16px standard gutter.
- **Elevation:** 0 (Flat design with subtle borders).
